```

│  [≡]  QuantTrader          🔍 Ctrl+K    🔔(3)  🌙  ⚙️        │  ← 頂部導覽列 (56px)
├────────┬────────────────────────────────────────────────────────┤
│        │                                                        │
│  📊    │              主要內容區域                                │
│ 儀表板  │              (Content Area)                             │
│        │                                                        │

│        │                                                        │
│  🤖    │                                                        │
│ 策略    │                                                        │
│        │                                                        │

│        │                                                        │

│ 設定    │                                                        │

└─────────────────────────────────────────────────────────────────┘
```

- **側邊欄**：寬 64px（收合）/ 240px（展開），深色背景
- **頂部欄**：固定不動，含搜尋、通知、主題切換
- **狀態列**：顯示連線狀態和最後數據更新時間

## 配色規範

| 用途 | 深色主題 | 淺色主題 |
|------|---------|---------|
| 背景色 | `#0f1117` | `#ffffff` |
| 側邊欄 | `#161b22` | `#f6f8fa` |
| 卡片背景 | `#1c2128` | `#ffffff` |
| 主要文字 | `#e6edf3` | `#1f2328` |
| 次要文字 | `#7d8590` | `#656d76` |
| 上漲/獲利 | `#3fb950` | `#1a7f37` |
| 下跌/虧損 | `#f85149` | `#cf222e` |
| 強調色 | `#58a6ff` | `#0969da` |
| 警告色 | `#d29922` | `#9a6700` |
| 邊框 | `#30363d` | `#d0d7de` |

## 字體規範

- **主字體**: Inter (UI), JetBrains Mono (程式碼)
- **指標數字**: 使用 tabular-nums 等寬數字特性，避免數字跳動
- **大小層級**:
  - 主要指標: 28px bold
  - 次要指標: 18px semibold
  - 正文: 14px regular
  - 輔助文字: 12px regular
